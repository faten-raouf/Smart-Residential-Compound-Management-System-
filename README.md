# Smart Residential Compound Management System

A comprehensive management system designed to simulate daily operations inside a smart residential compound. The system supports multiple user roles and components with specific permissions to ensure efficient management of compound services.

## 🛠️ Built With
- *Language:* C++
- *Concepts Used:* Object-Oriented Programming (OOP), Data Structures, and Database Design concepts.

## 👥 The 7 Core Classes & Features

### 1. Resident
- Register a new account with details (ID, Name, Phone, Email, Apartment Number, Password).
- View personal data and profile details.

### 2. Admin
- Manage and save system configuration data (ID, Name, Phone, Email, Password).
- Access and monitor compound statistics.

### 3. Technician
- Receive and manage assigned service requests (ID, Name, Specialization).
- Display technician status and workload.

### 4. Service Request
- Create maintenance requests with details (Request ID, Type: Plumbing/Electricity/Carpentry, Priority: High/Medium/Low, Description, Status).
- Track and update request status (Pending, Assigned, In Progress, Completed, Cancelled).

### 5. Facility
- Manage compound facilities (Facility ID, Name, Type, Availability).
- Check availability for reservations.

### 6. Complaint
- File complaints with clear records (Complaint ID, Description, Status, Date).
- Display details for admin review.

### 7. Booking
- Create and manage facility reservations (Booking ID, Date, Time, Status).
- Display reservation confirmation details.
