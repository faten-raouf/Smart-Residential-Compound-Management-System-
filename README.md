# Smart Residential Compound Management System

A comprehensive management system designed to simulate daily operations inside a smart residential compound. The system supports multiple user roles with specific permissions to ensure efficient management of compound services.

## 🛠️ Built With
- *Language:* C++
- *Concepts Used:* Object-Oriented Programming (OOP), Data Structures, and Database Design concepts.

## 👥 User Roles & Features

### 1. Resident
- Register a new account.
- Create maintenance and service requests (e.g., Plumbing, Electricity, Carpentry).
- File complaints with details and dates.
- Book compound facilities (e.g., Gym Sessions, Pool Slots, Event Halls).

### 2. Admin
- Manage and save system configuration data.
- Access and monitor all compound data and services.

### 3. Technician
- Receive and manage assigned service requests.
- Update request status (Pending, In Progress, Completed).
